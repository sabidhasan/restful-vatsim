# restful-vatsim

Serves RESTful data from VATSim flight simulation network

Used by VATSee on the backend.

# Libraries Required

- Flask (at least 0.12.2)
- flask_restful
- requests (at least 2.12.4)
- webargs (at least 1.8.1)
- StringDist (at least 1.0.9)
